# 코딩 테스트 합격자 되기 - 자바 편

**"코딩 테스트 합격자 되기(자바 편)"** 책의 문제 풀이를 정리한 레포지토리입니다.

## 프로젝트 구조

```
src/main/java/io/pinkspider/
├── array/    # 배열
├── stack/    # 스택
├── queue/    # 큐
├── hash/     # 해시
└── tree/     # 트리
```
